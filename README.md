## C# 

因為本篇有一些課程的東西，做個紀錄。

當初因為覺得開 VStudio 專案時將程式碼檔案拉出，所以造成非專案檔，只剩 cs 檔

我是使用 csc 製作 .exe 檔案

以下是新增 path 的教學

```
我的路徑如下：
C:\Windows\Microsoft.NET\Framework64\v4.0.30319
```

1. 我的電腦 or 本機 → 右鍵
2. 內容
3. 進階系統設定
4. 環境變數
5. 使用者變數(上) / 系統變數(下)
6. 那我們先新增 使用者變數 尋找 Path
7. 編輯 → 新增 → 貼上 路徑
