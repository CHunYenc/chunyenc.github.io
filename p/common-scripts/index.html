docker run -d \

-p 6379:6379 \
--name dev-redis \
--network dev-network redis

Q: 為什麼有 dev-network ?

A: 為了可以讓開發容器可以直接使用 dev-redis 進行連線 ! 前提是開發容器也要使用 dev-network 建立 !
